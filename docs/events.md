# College Events and Workshops

1. DBMS Crash Course Workshop:
   - Focus on SQL, joins, indexing, and query optimization.
   - Recommended for students scoring less than 15/25 in DBMS internals.

2. Operating Systems Lab Bootcamp:
   - Hands-on sessions on CPU scheduling and synchronization problems.

3. Coding Club DSA Series:
   - Weekly sessions on arrays, linked lists, stacks, queues, and trees.
   - Ideal for students who want to strengthen data structures foundations.
